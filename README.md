# REST Server Starter Project (DDD)

REST server 
* NodeJs
* Express
* MongoDb
* Typescript
* DDD

Extras:
* nodemon
* cors
* DotEnv
* Express Validators
* jest

## Usage:

* Create .env file and set (see .env.dist for more details):
    * Development port
    * Mongo Atlas connection URL
* Install node dependency packages: npm install
* Transpile typescript code using tsc
